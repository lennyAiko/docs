Internet is the inter connection of computer users. Internet serves as a medium for web. It is also a cable which connect computer users. The tool required to view a webpage is called a Browser. It is a software that renders web pages. Examples of web browsers are:

  • Mozilla firefox.
  • Opera mini.
  • UC-Browser.
  • Internet explorer.
  • Google chrome. etc.

Procedures for browsing

There are some procedures taken before being able to surf the internet with your browser. These steps are as follows:

  • Uniform Resource Locator[URL]: It is the code that will direct the client.
  • Internet Service Provider[ISP]: This is the corporation that provides the services for you to be able to connect to the internet.
  • Domain Name System[DNS] Server: This serves as a poster address containing all IP[Internet Protocol] address of all sites and checks the database.
  • Server: They are computer machine whose duty is to serve any request from the client-machine and it must be online 24/7.

Basics in the web

  • Hypertext Markup language[HTML]: It is used to structure a webpage and takes care of the layout.
  • Cascading Style Sheet[CSS]: It is used to style the webpage.
  • Javascript: It is used to add functionality to the webpageto begin some certain actions.

Structure of HTML tags

This is an example of a basic html structure.

Example explained:

  • The DOCTYPE declaration defines the document type to be HTML.
  • The text between <"html"> and <"/html"> describes an HTML document.
  • The text between <"head"> and <"/head"> provides information about the document.
  • The text between <"title"> and <"/title"> provides a title for the document.
  • The text between <"body"> and <"/body"> describes the visible page content.
  • The text between <"h1"> and <"/h1"> describes a heading.
  • The text between <"p"> and <"/p"> describes a paragraph.
